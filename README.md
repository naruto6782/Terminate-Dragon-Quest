# 🐉 Terminate-Dragon-Quest

> 💻 两个菜鸟的 C++ 大作业(仅支持windows系统)

---

🎮 **运行方式：**  
双击 `run.bat` 启动游戏！
如果直接点击build里的main.exe，部分符号可能出现乱码，请使用 `run.bat` 启动。

---

## 📦 项目简介

这是一个基于终端的回合制 RPG 游戏，灵感来源于《勇者斗恶龙》。你将扮演勇者，与怪物展开激烈的战斗！

---

## 💡 特性预览（可根据实际内容增删）

- 回合制战斗系统 ⚔️  
- 英雄 & 怪物属性系统 🧙‍♂️ 👾  
- 状态特效：中毒、防御、困倦等 💫  
- 终端图形表现增强 🌈
